# Poscash App

clone this repo
```
git clone https://github.com/AnwarAan/poscash
```

configure file .env First!

### Setup DB

```
myslq -u root -p
```
enter password
```
create database poscash;
```
check database
```
show databases;
```
```
exit;
```


### Run Client
```
cd ./client
```
```
npm install
```
```
npm run dev
```

### Run Server
```
cd ./server
```
```
npm install
```
```
nodemon index.js
```
